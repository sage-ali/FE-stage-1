# Task Card Component

This project implements a high-fidelity Task Card component, demonstrating frontend best practices, accessibility, and testability.

## Deployment to GitHub Pages

This project is deployed to GitHub Pages. You can view the live demo [here](https://sage-ali.github.io/FE-stage-1/).

To deploy this project yourself:

1.  Ensure you have `gh-pages` installed: `pnpm add -D gh-pages`
2.  Run the deploy script: `pnpm run deploy`

This will build the project and push the `dist` directory to the `gh-pages` branch of your repository.
