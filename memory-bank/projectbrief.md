# Project Brief: Task Card Component

## Overview

Build a high-fidelity Task Card component for a productivity app. The component must be interactive, accessible, and follow strict automated testing requirements using specific `data-testid` attributes.

## Core Requirements

- Clean, "alive" UI/UX.
- Semantic HTML implementation.
- Real-time updates for due dates and time remaining (every 30-60 seconds).
- Strict adherence to `data-testid` mapping.
- Mobile-responsive design.

## Technical Stack

- Vanilla JavaScript
- CSS (Custom Properties, Flexbox/Grid)
- HTML5 (Semantic elements)
